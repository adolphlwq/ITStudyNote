#Docker简介
Docker是很流行的容器技术。它主要有以下3个特点：
* 轻量
* 快速
* 弱安全

个人觉得`轻量`和`快速`是docker能够流行的主要原因。2014年自己就对Docker很感兴趣了，结果到2015.5才开始学习。这个笔记主要记录学习的总结和体会，方便以后复习。
